Yangın, sel, deprem gibi potansiyel tehlikeleri gerçek zamanlı olarak algılayarak, kullanıcıları çeşitli iletişim kanalları üzerinden anında bilgilendirir.</p>

### Kurulum Adımları

1. Bu depoyu bilgisayarınıza klonlayın:
   ```bash
   git clone https://github.com/eabaci42/32AUS.git
   cd 32AUS
   ```

2. Arduino IDE'yi açın ve ESP32 desteğini ekleyin:
   - `Dosya > Tercihler` menüsünden ek kartlar URL'sine şunu ekleyin:
     ```
     https://dl.espressif.com/dl/package_esp32_index.json
     ```
   - `Araçlar > Kart > Kart Yöneticisi` menüsünden ESP32 paketini yükleyin.

3. Gerekli kütüphaneleri yükleyin:
   - `Araçlar > Kütüphane Yöneticisi` menüsünden şu kütüphaneleri arayıp yükleyin:
     - ArduinoJson
     - UniversalTelegramBot
     - ADXL345
     - WiFiClientSecure
     - DHT sensor library

4. `src/communication/telegram.cpp` dosyasını açıp Telegram Bot Token'ınızı güncelleyin:
   ```cpp
   #define BOT_TOKEN "TELEGRAM_BOT_TOKEN_BURAYA"
   ```

5. `src/utils/other.cpp` dosyasında WiFi bağlantı bilgilerinizi güncelleyin:
   ```cpp
   wifiname = "WIFI_SSID";
   wifipassword = "WIFI_PASSWORD";
   ```

6. ESP32 kartınızı bilgisayarınıza bağlayın ve `src/aus-ertugrul.ino` dosyasını yükleyin.

## <div align="center">📊 Telegram Komutları</div>

<div align="center">
  <img src="https://media.giphy.com/media/3o6Zt9pLJJJcQK6YAE/giphy.gif" alt="Telegram" width="450" />
</div>

Telegram botu üzerinden şu komutları kullanabilirsiniz:

- `/start` - Bot ile etkileşim başlatır ve mevcut komutları listeler
- `/status` - Tüm sensörlerin mevcut durumunu gösterir
- `/heat` - Anlık sıcaklık değerini gönderir
- `/humidity` - Anlık nem değerini gönderir
- `/heathum` - Sıcaklık ve nem değerlerini beraber gönderir
